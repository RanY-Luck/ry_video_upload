# 监控小红书博主

这个工作流用于监控指定小红书博主的主页，一旦发现有新笔记发布，就会自动下载笔记的图片、视频和文案到本地，并可以通过 Bark 推送通知。

## 准备工作

1. 确保已经安装了脚本运行所需的依赖（`pip install playwright httpx python-dotenv`）。
2. 需要预先配置好环境，包括 Playwright 的 Chromium 浏览器（`playwright install chromium`）。
3. 确保 `.env` 文件中配置了相应的设置，或者在运行命令时手动指定。

## 运行监控脚本

你可以通过以下命令来启动监控：

```bash
// turbo
python xhs_monitor.py --users "博主主页链接"
```

*把 `博主主页链接` 替换为你想监控的博主的真实主页链接，例如 `https://www.xiaohongshu.com/user/profile/xxx`*

### 可选参数

*   `--interval 300`: 自定义检查间隔（单位：秒，默认 600 秒）。
*   `--once`: 仅检查一次，不循环执行。

### 首次运行注意事项

第一次运行时，脚本会打开一个浏览器窗口：
1. **如果页面要求登录：** 请在弹出的浏览器窗口中**手动完成扫码或短信登录**，以及可能的滑块验证。脚本会等待最多 60 秒。
2. **免登录持久化：** 登录成功后，脚本会自动识别并保存登录状态（Cookie 等）到 `downloads/xhs_monitor/.xhs_storage_state.json`。后续再次运行将实现免登录。
3. **基线记录：** 第一次成功运行会记录该博主当前的所有笔记作为“已知”，不会立即下载它们。只有以后他在监控期间发布的新内容才会被下载。
