# 上传内容到小红书

这个工作流用于将本地下载或准备好的图文/视频内容自动上传发布到你自己的小红书账号中。通常与博主监控脚本配合使用。

## 准备工作

1. 确保已经安装了脚本运行所需的依赖。
2. 确保本地有待上传的图文或视频素材。

## 运行上传脚本

运行上传脚本：

```bash
// turbo
python xhs_upload.py
```

*(注意：你可能需要根据实际情况修改上方的命令，指定具体的运行参数或配置文件)*

### 上传配置项（根据实际脚本文档修改）

*   请确保已经配置了你要发布的账号的登录状态。第一次运行可能需要在弹出的浏览器中手动登录。
*   确认上传脚本中设置的**待发布素材目录**是否正确指向了你准备好的内容夹。
*   为了防止重复发布，确保脚本有记录已发布历史的机制。

### AI 润色功能

如果脚本支持 AI 内容润色（例如使用 qwen-turbo），请确保相关的 API Key（如百炼平台的 `DASHSCOPE_API_KEY`）已经在环境变量或 `.env` 文件中正确配置。
