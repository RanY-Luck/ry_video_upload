# 独立视频号上传工具 - 人工审核版

## 功能说明

支持人工审核的视频号上传工具,AI 生成标题和标签后,您可以手动修改审核,确保内容准确无误。

## 工作流程

### 第一步: AI 生成元数据

程序会为每个视频生成一个同名的 `.txt` 文件,包含:
- AI 分析生成的标题
- AI 分析生成的标签

例如:
```
视频文件: video1.mp4
元数据文件: video1.txt
```

### 第二步: 人工审核修改

打开生成的 `.txt` 文件,根据实际视频内容修改标题和标签。

**文件格式示例:**
```
标题: 🔥超可爱的小猫咪日常,萌化你的心!
标签: 萌宠,猫咪,可爱,日常,治愈,宠物,推荐,必看

# ========== 使用说明 ==========
# 第一行是标题 (格式: 标题: xxx)
# 第二行是标签 (格式: 标签: tag1,tag2,tag3)
# 请根据视频内容修改标题和标签
# 修改完成后保存文件即可
# ==============================
```

**修改要点:**
- 标题要与视频内容相符
- 标签要准确描述视频内容
- 保持格式不变 (`标题:` 和 `标签:`)
- 标签用英文逗号分隔

### 第三步: 确认上传

修改完所有文件后,在控制台按回车键,程序会:
1. 读取您修改后的标题和标签
2. 自动上传到视频号
3. 上传成功后删除视频和元数据文件

## 使用方法

### 快速启动
双击运行 `run_upload.bat`

### 命令行运行
```bash
python standalone_upload.py
```

## 配置说明

在 `standalone_upload.py` 中可修改:

```python
# AI API Key (必须配置)
DASHSCOPE_API_KEY = "your-api-key-here"

# 视频分类
CATEGORY = TencentZoneTypes.CUTE_PETS.value

# 上传后是否删除视频
DELETE_AFTER_UPLOAD = True
```

## 注意事项

1. **AI 生成仅供参考**: AI 可能生成不准确的内容,请务必人工审核
2. **保持格式**: 修改时保持 `标题:` 和 `标签:` 的格式
3. **标签分隔**: 标签用英文逗号分隔
4. **文件编码**: 确保 .txt 文件使用 UTF-8 编码保存
5. **已存在的文件**: 如果 .txt 文件已存在,程序不会重新生成,会使用现有文件

## 常见问题

### Q: 如何重新生成某个视频的元数据?

A: 删除对应的 `.txt` 文件,重新运行程序即可。

### Q: 可以手动创建 .txt 文件吗?

A: 可以,只要保持格式正确即可。

### Q: 上传失败的视频会怎样?

A: 上传失败的视频和元数据文件不会被删除,可以修改后重新运行。

### Q: 如何跳过某个视频不上传?

A: 删除对应的 `.txt` 文件或将视频移出 `Upload/videos/` 目录。

### Q: 可以批量修改标题和标签吗?

A: 可以使用文本编辑器的批量查找替换功能,或编写脚本批量处理 .txt 文件。

## 技巧建议

1. **观看视频后再修改**: 建议先观看视频,了解内容后再修改标题和标签
2. **参考平台热门**: 可以参考视频号平台的热门内容,优化标题和标签
3. **保持一致性**: 同类型视频可以使用相似的标签,便于用户发现
4. **标题吸引力**: 标题要有吸引力,但不要夸大或误导
5. **标签数量**: 建议 8-15 个标签,覆盖视频的主要内容和相关话题

## 安全提示

- 上传成功后会自动删除本地文件,请确认已备份
- 如需保留原文件,设置 `DELETE_AFTER_UPLOAD = False`
